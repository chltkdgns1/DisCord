# 백준

</hr>

### 단, 실시간으로 업뎃되는 것은 아니기 때문에 다른 추가 명령어인 !업데이트가 필요합니다.

# 강화

</hr>

### 강화는 아이템을 1번부터 100번까지 연속으로 강화시킬 수 있습니다.
### 한번에 강화할 수 있는 이유는 사용자들의 의견이 나와서 수용했습니다.
